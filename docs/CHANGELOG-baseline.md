# Zukunftsvorschau (Roadmap)

Hinweis:
- Diese Datei ist eine Vorschau auf kommende Features.
- Der echte Release-Verlauf steht in `docs/CHANGELOG.md`.
- Versionsschema gemaess `versionsverlauf.md`: `0.0.x` -> `0.1.x` -> `0.2.x`, Hotfix als Buchstabe (`0.1.1a`).

## Naechste Ausbaustufen

### v0.1.1 - Strom-Erweiterungen
Geplante Features:
- Erweiterte Strom-KPIs (Tag/Woche/Monat/Jahr) mit klarer Laien-Sprache
- Bessere Erklaertexte direkt in der Karte
- Feinere Steuerung der angezeigten Bereiche (einfach vs. erweitert)

### v0.1.1a - Hotfix-Linie
Geplant fuer:
- schnelle Fehlerbehebungen ohne neue Features
- Stabilitaet nach `v0.1.1`

### v0.2.0 - PV/BKW-Ausbau
Geplante Features:
- PV-Erzeugung, Einspeisung, Eigenverbrauch in eigener Sektion
- Optionale Kosten- und Amortisationsberechnung
- Modul-Schalter, damit Einsteigeransicht schlank bleibt

## Weiterer Ausblick (nach v0.2.0)
- Erweiterung auf Gas
- Erweiterung auf Wasser
- Einheitliches Energie-Dashboard (Strom/Gas/Wasser)
